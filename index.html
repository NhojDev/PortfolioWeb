Software Projects

Bombshell

Bombshell is a dating sim made for the Tavern Game Jam - Winter '23 that was developed in Godot by my friends and I. The games was inspired on the idea of "bombing" a date and transformed into a more literal sense. The gameplay consists of managing depleting bars in which the player has to manage in order to have a success ending.

Click here or the image for the repo.

Cracked

Cracked is a video game prototype built in Godot, designed by Naton Cai and I. The game is intended to in a smash style PVP game developed on the theme of smashing vases similar to video games such as Legend of Zelda. The game was designed using Godot and most of the visual assets and animations were created by me.

Click here or the image for the repo.

Robot Finds Kitten

This project was about adding a new game experience to Robot Finds Kitten by Leonard Richardson and it was primarily done in Python3. Right off the bat, I wanted to make the game experience more of a grind and have a little bit of a mystery vibe to Robot Finds Kitten so I added a fog around the player to make it harder to see all the objects. I also added items that basically don't belong among the other items so try and create a sense of mystery to the game.

Click here or the image for the repo.

FTP Client

This a class project, where I had to work with a team to create a FTP client. The client was mainly built using Python and it only has some basic functionalities such as uploading and downloading from the FTP server.
